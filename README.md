## 📝 Day01
> ### JSP: Java Server Page(html): 웹언어
> 자바 코드에서 html 작성이 되는 것이 Servlet(서블릿)이다.  
> 서블릿은  "" 안에 "", 태그, 인라인 속성 등 해주어야 할 게 너무 많아서  
> 오타 발생률이 높고 불편하며 가독성이 좋지 않다.  
> 그런 부분을 보완해서 나온 것이 JSP이다.  
> **html에서 java 작성이 가능**한 것!
